# 📜 Kerygma Theology AI Lab (v2.7) 빠른 시작 가이드

**Kerygma Theology AI Lab**에 오신 것을 환영합니다! 이 가이드는 프로그램 설치부터 첫 검색까지 초보자도 쉽게 따라 할 수 있도록 안내합니다.

---

## 1단계: 압축 풀기
다운로드한 `Kerygma_Theology_AI_Lab_v2.7.23_Clean.zip` 파일의 압축을 해제합니다.
> [!IMPORTANT]
> 압축을 푼 폴더는 앞으로 서재 데이터가 저장될 장소입니다. 가급적 **'문서'**나 **'사용자 폴더'** 등 편리한 곳에 보관해 주세요.

---

## 2단계: 자동 설치 (최초 1회)
사용 중인 컴퓨터 운영체제에 맞는 설치 파일을 실행합니다.

### 🍎 Mac 사용자
1. 폴더 안의 **`1_INSTALL_MAC.command`** 파일을 찾습니다.
2. **보안 안내**: 그냥 더블클릭하면 "확인되지 않은 개발자" 메시지가 뜰 수 있습니다. 
   - 파일 위에서 **[마우스 오른쪽 버튼 클릭]** -> **[열기]**를 선택하세요.
   - 나타나는 창에서 다시 한번 **[열기]**를 누르면 터미널 창이 열리며 설치가 시작됩니다.
3. 설치가 완료될 때까지 터미널 창을 닫지 마세요. (약 2~5분 소요)

### 🪟 Windows 사용자
1. 폴더 안의 **`1_INSTALL_WIN.bat`** 파일을 더블클릭합니다.
2. 검은색 화면이 나타나며 필요한 라이브러리를 자동으로 설치합니다.

---

## 3단계: 연구소 실행
설치가 완료되었다면 이제 연구소를 실행할 차례입니다.

1. **`3_START_MAC.command`** (맥) 또는 **`3_START_WIN.bat`** (윈도우) 파일을 실행합니다.
2. 잠시 기다리면 **인터넷 브라우저**가 자동으로 열리면서 연구소 화면이 나타납니다.

---

## 4단계: 초기 설정 (중요!)
연구소를 제대로 사용하려면 AI의 '열쇠'인 API 키가 필요합니다.

1. 화면 왼쪽 메뉴에서 **`⚙️ 설정`** 탭을 클릭합니다.
2. **API 키 (선택 사항)**: 보유하고 계신 API 키(Claude, GPT, Gemini 등)를 입력합니다.
   - **로컬 모드 (키 미입력)**: API 키 없이도 내 서재의 인덱싱과 실시간 검색은 무료로 무제한 사용 가능합니다.
   - **AI 모드 (키 입력 시)**: 검색 결과에 대한 'AI 분석 리포트'와 '심층 요약' 기능을 사용할 수 있습니다.
   - 키 발급: [Anthropic](https://console.anthropic.com/) | [OpenAI](https://platform.openai.com/api-keys) | [Google Gemini](https://aistudio.google.com/app/apikey)
3. 바로 아래 **`OBSIDIAN VAULT`** 항목에는 연구 결과를 저장할 옵시디언 볼트 폴더 경로를 입력해 주세요. (추천)
4. 하단의 **`💾 설정 저장`** 버튼을 꼭 눌러주세요!

---

## 5단계: 첫 서재 업데이트 및 검색
1. **파일 추가**: `01_Library/inbox/` 폴더에 분석하고 싶은 PDF 파일을 넣습니다.
2. **인덱싱**: 메뉴에서 **`📤 파일 업로드`**를 클릭하고 **`🚀 인덱싱 시작`** 버튼을 누릅니다. (AI가 책을 읽고 기억하는 과정입니다.)
3. **검색**: 메뉴에서 **`🔍 검색`**으로 돌아가 질문을 던져보세요! 
   - 예: "바르트의 칭의론에 대해 설명해줘"

---

## 6단계: 옵시디언(Obsidian) 연동 (선택 사항)
본 프로그램을 옵시디언 내부 창에서 바로 사용하고 싶다면 다음 설정을 진행하세요.

1. 옵시디언 **커뮤니티 플러그인**에서 **`Custom Frames`**를 검색하여 설치하고 [활성화]합니다.
2. 플러그인 설정에서 **`add-frame`**을 클릭하여 새 프레임을 등록합니다.
   - **Name**: `Kerygma Lab` (원하는 이름)
   - **Url**: `http://localhost:8501`
   - **Icon**: `library` 또는 `search`
   - **Show on Ribbon**: **활성화(ON)** (이 설정을 켜야 왼쪽 사이드바에 아이콘이 나타납니다.)
3. 설정 완료 후 옵시디언 왼쪽 사이드바에 생성된 아이콘을 누르면, 옵시디언 안에서 연구소를 바로 이용할 수 있습니다.
