# 📜 Theology AI Lab v3.0.0 - 빠른 시작 가이드

**Theology AI Lab**에 오신 것을 환영합니다! 
신학 PDF를 업로드하고, AI 기반 의미 검색으로 연구하세요.

---

## 1️⃣ 압축 풀기

다운로드한 `Theology_AI_Lab_v3.0.0_E5base.zip` 파일의 압축을 해제합니다.

> 💡 압축을 푼 폴더는 앞으로 서재 데이터가 저장될 장소입니다. 
> **문서** 또는 **사용자 폴더** 등 편리한 곳에 보관하세요.

---

## 2️⃣ 설치 (최초 1회)

### 🍎 Mac
1. **`1_INSTALL_MAC.command`** 파일을 **우클릭 → 열기**
2. 보안 경고가 뜨면 **열기** 클릭
3. 터미널 창에서 설치 진행 (약 2~5분)

### 🪟 Windows
1. **`1_INSTALL_WIN.bat`** 파일을 더블클릭
2. 검은 화면에서 자동 설치 진행

> ⚠️ **Python 3.11**이 설치되어 있어야 합니다.

---

## 3️⃣ 실행

### 🍎 Mac
**`3_START_MAC.command`** 더블클릭

### 🪟 Windows
**`3_START_WIN.bat`** 더블클릭

✅ 자동으로 브라우저가 열리고 **http://localhost:8501** 에 접속됩니다.

---

## 4️⃣ 초기 설정

1. 화면 왼쪽 **⚙️ 설정** 탭 클릭
2. **API 키 등록** (선택):
   - 없어도 인덱싱/검색 가능 ✅
   - 있으면 AI 분석 리포트 사용 가능
   - 지원: Gemini, OpenAI, Anthropic
3. **Obsidian Vault 경로** 입력 (선택)
4. **💾 설정 저장** 클릭

---

## 5️⃣ 첫 검색

1. **PDF 추가**: `01_Library/inbox/` 폴더에 PDF 넣기
2. **인덱싱**: 📤 파일 업로드 → 🚀 인덱싱 시작
3. **검색**: 🔍 검색 탭에서 질문하기
   - 예: "바르트의 칭의론에 대해 설명해줘"

---

## 🔍 검색 팁 (하이브리드 검색)

두 가지 검색이 자동으로 함께 작동합니다:

| 검색 유형 | 아이콘 | 언제 사용? |
|----------|-------|----------|
| **의미 검색** | 🧠 | 개념/주제 검색 ("칭의론", "은혜") |
| **키워드 검색** | 🔤 | 원어/음역 검색 (hesed, ἀγάπη) |

### 검색 예시

```
"바르트의 칭의론" → 🧠 의미로 관련 내용 검색
"hesed" → 🔤 히브리어 음역 정확 매칭
"Gnade" → 🧠+🔤 둘 다 작동
```

> 💡 **팁**: 히브리어/헬라어는 영문 음역으로 검색하면 더 정확해요!
